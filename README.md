## Gith Cheat Sheet

- git init: Inicializa un novo repositorio de Git no directorio actual.

- git clone: Clona un repositorio existente a un directorio local.

- git add: Engade cambios á "zona de preparación" (staging area) para que estean listos para ser confirmados.

- git commit: Confirma os cambios na "zona de preparación" e agrégaos ao historial do repositorio.

- git status: Mostra o estado actual do repositorio, incluíndo cales arquivos foron modificados e cales están preparados para ser confirmados.
- 
- git log: Mostra o historial de confirmacións (commits) do repositorio.

- git diff: Mostra as diferenzas entre as versións actual e anterior dun arquivo.

- git branch: Mostra unha lista de ramas existentes no repositorio.

- git checkout: Cambia entre ramas ou versións de arquivos.

- git merge: Combina cambios dunha rama con outra.

- git pull: Trae os cambios máis recentes dun repositorio remoto.

- git push: Envía os cambios locais a un repositorio remoto.
